ESP32-C3 와 Android 앱 사이 Bluetooth 기반 MOSFET 으로 DC 전원을 제어합니다.

아래 사진 속 부하는 COB(Chip On Board) LED 입니다.

 

- 수동 ON/OFF 제어

- Bluetooth 장치 검색 / 등록 / 삭제

- 장치 원격 ON/OFF 제어

- 수동 ON/OFF 상태 Android 앱 동기화

- Bluetooth 온라인 LED (점멸: 연결 중, ON: 연결됨)
