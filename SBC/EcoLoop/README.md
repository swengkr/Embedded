라즈베리파이 SBC(Single Board Computer) 와 PIC MCU를 활용한 저전력·무중단·지속 운용 시스템을 제작하여 운영 중입니다.

### 구성 요소
- 컴퓨팅 모듈 : 라즈베리파이1 모델 B+
- Wakeup : PIC 12F675
- DC 전원 제어 : MOSFET
